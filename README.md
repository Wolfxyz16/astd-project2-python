### Project structure
* `data`: directory to store all the original csv data
* `esrnn_updated`: [TODO]
* `main.py`: entry of the main application
* `generate_equations.py`: use this file after predicting with the symbolic regression model, it generates all the calculated equations as png file
* `outputs`: PySR outputs folder, not important
* `plots`: after loading the time series, we plot them and store their descomposition here
* `pysr_equations`: directory where the equations found by the symbolic regression model
* `requirements.txt`: packages to install
* `results`: the forecast of our models is stored here as csv files
* `src`: directory where we can find the files related with our models
